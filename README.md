#  Task Tracker App

Приложение для управления задачами с поддержкой повторений, подзадач и напоминаний.

---

##  Возможности

* Создание и редактирование задач
* Повторяющиеся задачи (ежедневно, еженедельно, ежемесячно, ежегодно, курс)
* Подзадачи
* Напоминания с настройкой времени
* Выбор звука уведомлений
* Отображение задач:

  * День
  * Месяц
  * Год

---

##  Архитектура

Приложение построено с использованием:

* MVVM
* Clean Architecture

### Слои:

* **Presentation** — UI (Jetpack Compose, ViewModel)
* **Domain** — бизнес-логика, use-cases
* **Data** — Room, repository

---

##  Поток данных

```
UI → ViewModel → UseCase → Repository → DAO → Database
```

---

##  Технологии

* Kotlin
* Jetpack Compose
* Room
* Coroutines
* Flow

##  Цель проекта

Практика разработки Android-приложения с чистой архитектурой и сложной логикой задач.
